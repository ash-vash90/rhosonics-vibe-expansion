---
name: No fake industrial chrome
description: Forbidden patterns when trying to make UI feel "industrial"
type: constraint
---
NEVER add decorative spec-sheet metadata to make something look industrial:
- No fake "REV / STATUS / SCOPE" strips
- No fabricated "[ 01 / 02 ]" counters as chrome
- No dashed dividers, ticks, or rule bars added for vibe
- No "> previously" terminal markers
- No random uppercase mono labels next to headings

Industrial feeling comes from: precise typography hierarchy, restraint, real information density, accurate alignment, and meaningful structure — NOT from decorative dividers, labels, or text fragments invented to look technical.

**Why:** The user explicitly rejected this as "random text and dividers for no reason." Decoration that does not carry information violates the Deliberate Elements principle.
