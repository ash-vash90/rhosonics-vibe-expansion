name: JetBrains Mono usage rules (hard)
description: JetBrains Mono is for short uppercase data only — never prose
type: constraint
---
JetBrains Mono (font-data) rules — NO EXCEPTIONS:
- ONLY for short data labels, numerals, codes, units
- ALWAYS uppercase (tracking ~0.14em+)
- NEVER for paragraph text
- NEVER for headings or titles
- NEVER for sentence-case or lowercase text
- NEVER for quotes, descriptions, change notes, body copy

Prose uses Instrument Sans (font-ui). Headings use Instrument Sans.

**Why:** User explicit rule. Mono in prose looks like a code block, not industrial design.
